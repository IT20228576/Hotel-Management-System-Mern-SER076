## Getting Started

To run project follow these simple example steps.

### Installation

### Backend side

1. Go to backend folder
   ```sh
   cd backend
   ```
2. Install dependencies
   ```sh
   npm install
   ```
3. Start the server
   ```sh
   npm start
   ```

### Frontend side

1. Go to backend folder
   ```sh
   cd frontend
   ```
2. Install dependencies
   ```sh
   npm install
   ```
3. Start the server
   ```sh
   npm start
   ```
4. Default Server running at http://localhost:1234/
